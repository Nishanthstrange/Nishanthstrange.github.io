# Interactive 3D Software Developer Portfolio

A responsive, high-performance portfolio website built to showcase software engineering projects, data structures, and machine learning UI implementations.

## 🚀 Live Demo
View the live site here: [https://nishanthstrange.github.io](https://nishanthstrange.github.io)

## 🛠️ Technical Stack
- **Frontend Core:** HTML5, Tailwind CSS, JavaScript (ES6+)
- **Graphics Engine:** Three.js (WebGL 3D Particle Constellation)

## 📂 Featured Projects
- **Smart Attendance System:** Digital tracking and pattern analysis using historical data trends.
